# 📋 Техническое задание: Распределение заказов в ИФЗ

## 📌 Общая информация

| Раздел | Описание |
|--------|----------|
| **Проблема** | После передачи заказа в ИФЗ, менеджер работает в базе ИФЗ и руками делит заказ на несколько заказов в рамках одного склада. Теряется цепочка связей заказов, документ не создается в УТ |
| **Задача** | Создать рабочее место для компоновки товара перед созданием заказа поставщику с выводом предварительного списка создаваемых документов |
| **Решение** | Выделить промежуточную форму для графического отображения и сортировки товара на несколько заказов поставщику с автоматическим распределением |

---

## 🎯 Постановка задачи

| Действие | Описание |
|----------|----------|
| **Кнопка** | Добавить на форму заказа клиента кнопку "Сформировать заказы" |
| **Открытие** | По нажатию показывается форма распределения товаров |

---

## 🖥️ Структура формы

| Элемент | Описание |
|---------|----------|
| **Шапка** | Заказ клиента (номер, дата) |
| **Кнопки управления** | Завершить / Очистить |
| **Поле ввода** | Введите объем ____ л |
| **Сворачиваемый блок** | Опции (текущие настройки складов) |

## 🔄 Алгоритм работы

| Шаг | Действие |
|:---:|----------|
| **1** | В заказ клиента добавляем кнопку с командой "Распределить заказ" |
| **2** | По нажатию показывается форма распределения |
| **3** | В форму загружается заказ клиента и настройки распределения складов из регистра |
| **4** | Заказ клиента остается в резерве на момент операций |
| **5** | Обновляем склады (колонка отправитель) в соответствии с настройками |
| **6** | В ТЧ формы выводим только идентификацию номенклатуры и МГХ |
| **7** | В Окно 1 выводим строки, сгруппированные по Склад + СкладИФЗ с агрегацией количества и объема |
| **8** | Окно 2 содержит распределенные грузоместа (будущие заказы поставщику) |
| **9** | Окно 3 — нераспределенные товары выделенной строки из Окна 1 |
| **10** | Окно 4 — распределенные товары выделенной строки из Окна 1 |

---

## 👤 Порядок действий для менеджера

| № | Действие |
|:--:|----------|
| **1** | Открыть заказ, нажать кнопку "Сформировать заказы" |
| **2** | Проверка: заполнены склады ИФЗ, нет созданных проведенных заказов поставщику |
| **3** | Форма автоматически заменяет склады по настройкам (можно изменить вручную) |
| **4** | В Окне 1 видит структуру: группировка Склад + СкладИФЗ |
| **5** | Выбрать строку в Окне 1, создать грузоместа в Окне 2 |
| **6** | В Окнах 3 и 4 заполнить грузоместа товарами |
| **7** | По завершению нажать кнопку "Завершить" |

## ⚙️ Команды и их логика

### 🔹 Завершить

| Проверка | Действие |
|----------|----------|
| ✅ Проверка распределенных товаров | Если есть нераспределенные → предупреждение |
| 🔄 Транзакция | Изменить склады в заказе клиента, провести заказ |
| ✅ Проверка доступного количества | С учетом резерва текущего заказа, если нет → откат |
| 📄 Создание заказов | Создать заказы поставщику по количеству грузомест |
| 📋 Вывод | Список ссылок на созданные заказы |

---

### 🔹 Распределить по объему

| Проверка | Действие |
|----------|----------|
| ✅ Проверка поля "Введите объем" | Если не заполнено → предупреждение |
| ✅ Проверка объема номенклатуры | Если не заполнен → предупреждение, остановка |
| 🔄 Алгоритм | Создать грузоместа и заполнить товарами по объему |
